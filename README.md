## 🚀 Quick Start

### Prerequisites

- **Node.js** 18+
- **Firebase** project with Auth + Firestore enabled
- **OpenRouter** API key

### Installation

```bash
# Clone the repository
git clone https://github.com/gpsaarvin/iTubeLearn.git
cd iTubeLearn

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
```

### Environment Variables

Create a `.env.local` file:

```env
# Firebase
NEXT_PUBLIC_FIREBASE_API_KEY=your_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_domain
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_bucket
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender
NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id

# OpenRouter
NEXT_PUBLIC_OPENROUTER_API_KEY=your_key
```

### Run

```bash
npm run dev        # Development (http://localhost:3000)
npm run build      # Production build
npm start          # Start production server
```

## 🔒 Content Safety

iTube Learn includes a built-in content filter that blocks searches for:
- 🚫 Adult/explicit content
- 🚫 Drug & substance abuse
- 🚫 Violence & weapons
- 🚫 Hacking & piracy
- 🚫 Gambling & scams

Filtering is enforced at **3 layers**: client-side UI, server API routes, and the roadmap generation page.
